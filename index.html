Projeto 1

O projeto "To-Do List" é uma aplicação web que permite gerenciar tarefas diárias de forma simples e eficiente. Os usuários podem adicionar, editar, remover e marcar tarefas como concluídas, além de incluir descrições detalhadas para cada uma. As tarefas são armazenadas localmente, garantindo que as informações sejam mantidas após o fechamento do navegador. Este projeto destaca habilidades em JavaScript e manipulação de DOM, visando melhorar a organização e a produtividade.Tecnologias: HTML, CSS, JavaScript.
